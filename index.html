Enter BEM!

What is BEM?

BEM - The B, E and M

  Description
Block Standalone entity that is meaningful on its own.
Element A part of a block that has no standalone meaning and is semantically tied to its block.
Modifier A flag on a block or element. Use them to change appearance or behavior.

Example: Footer HTML

			<footer class="footer">
			  <ul class="footer__navigation">
			    <li class="footer__navigation-item"></li>
			  </ul>
			  <p class="footer__disclaimers">...</p>
			</footer>

Example: Footer CSS

			.footer {}
			.footer__navigation {}
			.footer__navigation-item {}
			.footer__navigation-item--highlight {}
			.footer__disclaimers {}

Things to note

  1. Flat class structure
  2. Clear & consistent naming
  3. Clash resistant
  4. Specificity Wars resistant
  5. No use of tag names
  6. No use of #id's
  7. Facilitates components/modules -> reuse

Example: Footer LESS/SASS

			.footer {
			  &__navigation {
			    &-item {
			      &--highlight {}
			    }
			  }
			  &__disclaimers {}
			}

BEM Rules

  1. Structure: .my-block__my-element
  2. With modifier: .my-block__my-element--mofifier
  3. No more than one element allowed
  4. Lower case
  5. - between words
  6. __ between block & element
  7. -- for modifiers

Modifiers

			<button class="button">Click me</button>
			<button class="button button--danger">Don't click me</button>
			<li class="footer__navigation-item
			           footer__navigation-item--size-large
			           footer__navigation-item--highlighted"></li>

Figure this out: Easy

HTML 
			<footer class="footer">
			    <p class="footer__disclaimer">What color?
CSS 
			.footer__disclaimer { color: green; }

Debugging this...

			<li class="footer__navigation-item
			           footer__navigation-item--highlighted"></li>

...is easy

  1. Open file footer.less
  2. Go to navigation-item element styles
  3. That's it (no clashes, overrides, side-effects etc)

Not safe to edit:

			.description {
			    color: red;// ARGH!
			}

Safe to edit this:

				.footer__navigation-item--highlighted {
				    color: red; // Safe
				}
